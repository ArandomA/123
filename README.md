#### 依赖环境
1.  Hadoop
2.  Zookeeper
3.  HBase
4.  Oracle

#### 使用说明

1、 controller：页面访问控制部分，控制service执行mapper，封装数据并写入Session。
2、mapper：绑定SQL语句，连接Phoenix；
3、domain：存储实体类；
4、service：调用mapper；
